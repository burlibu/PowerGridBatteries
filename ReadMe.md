# Create: Power Grid Batteries

![Thumbnail](powergrid-batteires-thumbnail.png)
Add-on for the Create: Power Grid Mod.
My first mod/addon 
Since I found the powrgrid batteries 
Adds 4 batteries.

Note that the mod works fine with PowerGrid 0.5.5.1.
In 0.5.5.1 PowerGrid battery and small battery may seem to have similar specs, but total capacity is vastly different (120 Wh vs 1000 Wh).
Please backup your world before using it and report any issues.

## 🛠️ Requirements

- **Minecraft**: `1.21.1`
- **Mod Loader**: `NeoForge 21.1.231+`
- **Dependencies**:
  - `Create` (`6.0.10+`)
  - `Create: Power Grid` (`0.5.5.1+`)
